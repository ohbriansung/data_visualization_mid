Brian's D3 Visualization 1

Interpretation

Neighborhood axis presents different areas within San Francisco which was extracted from "Neighborhooods - Analysis Boundaries" column in the original file.
Weekday axis presents different days of a week which was calculated and extracted from "Call Date" column in the original file.
The color of D3 implementation is different from the prototype's since Tableau and D3 have such different sequential color encoding for red.

Interactivity

1. Hover over each cell in the heatmap will give you details of that cell.
2. Hover over each weekday will give you ability of comparing between each neighborhood on that day.
3. Hover over each neighborhood will give you ability of comparing between each day in that neighborhood.

Brian's D3 Visualization 2

Interpretation

Paramedic axis presents whether there was a paramedic on scene or not which was extracted from "ALS Unit" column in the original file.
Year axis presents the year when incidents happened which was calculated and extracted from "Call Date" column in the original file.
Call Type presents the incident types which was extracted from "Call Type" in the original file.
Neighborhood was used in this chart as well but not presented which was the same attribute as heatmap above.

Interactivity

1. Hover over each arc in pies will give you details of that cell.
2. Hover over each arc in pies will give you perpectives of that call type between each year and with/without paramedic.
3. Hover over each call type in legend will give you perpectives of that call type between each year and with/without paramedic.
